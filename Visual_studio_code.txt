SUBLIME TEXT

Es un editor de texto para escribir codigo en casi cualquier formato de archivo .
pensado desde un inicio en la velocidad haciéndolo uno de los editores de texto mas rapido y facil de usar.

Ademas sublime tex es una multiplataforma disponible para linux windows y XOS , sin embargo es un programa de pago , se puede descargar una versión de pruebas por un tiempo limitado…

--Caracteristicas

-Simplicidad posee una interfaz limpia y sencilla que no  muestra barras de herramientas y dialogos de configuración. 

-Minimapa permite la sencilla navegacion dentro del programa .

-DerecionadorPuede fijar colores para tu código dependiendo del lenguaje que estés empleando.

-amplia sintaxis  Soporta un grna numero de lenguajes (C,C++,C#, CSS,Python, Javascrip,Java , HTML.)

-Autocompletado  , ofrece una lista de palabras posibles de acuerdo a palabras que hayas empelado anteriormente 

Asimismo , sublime tex editor , posee una característica de resaltado para HTML , CSS, JAVA, PHP y muchos otros lenguajes .
